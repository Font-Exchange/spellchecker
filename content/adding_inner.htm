Dictionary

 

Adding dictionaries

You can install other dictionaries as necessary.

  1. Click Spell Checker Add-on (right content page)


     
  2. An installation page will appear


     
  3.  Select newer version and click install
     
  4. The extension file auto download and ready to install


     
  5. Click the Add to Firefox button on the page that opens. When the warning panel appears, clickAdd to allow the installation.

Switching dictionaries

  1. After the dictionary is installed, you can switch to it.  To switch to a different installed dictionary, Right-click in a multi-line field, select Languages, then select the odia dictionary you want to use.  

How do I enable dictionary

  1. Click Add-on on in Spell Checker panel
  2. Select add dictionary
  3. then ok
  4. The Dictionaries & Language Packs both installed
  5. Go to Spell Check Home
  6. Right Click text area
  7. Select Language
  8. Select Odia

Why is my spell Check not working

  1. 1. The spelling checker is enabled if [X] "Check Spelling" in the right -click context menu.
  2. Spell Checker has two mode. one is (Regional) and other is Globally.  When you press left mouse pointer in red colour misspell word, the suggestion word show within a box. If you right click the underline misspell  word suggestion word appear.
  3. Adding facility not in Regional mode
  4. Globally mode has adding facilities

Disabling automatic spell checking [Firefox]

  1. Click the menu button and select Settings.
  2. In the General panel, go to the Language section.
  3. Uncheck Check your spelling as you type.

 

Correcting misspelled words

  1. When spell-checking is enabled, you can easily correct misspelled words. To correct the misspelled word, Right-click on it and select one of the suggested words at the top of the menu.

 

If none of the suggested words is appropriate, you must manually edit it. If you are sure the word is spelled correctly, you can add it to the dictionary. To add a word to the dictionary, Right-click on it, and select Add to Dictionary.
